Results Snapshot

Key Finding 1: Casual users ride significantly longer on weekends compared to members.

Key Finding 2: The average trip duration for annual members is 45% lower than that of non-members.

Recommendation: Implement a weekend-specific marketing campaign targeting casual riders with membership incentives.
